# User Manager Node MySQL

Projeto de estudo para praticar desenvolvimento full stack com:

- HTML
- CSS
- JavaScript
- Node.js
- Express
- MySQL

## Objetivo

Criar um sistema CRUD de usuários, entendendo o caminho completo dos dados:

Tela HTML → JavaScript → Backend Express → Banco MySQL → Resposta na tela.

## Funcionalidades iniciais

- Estrutura de frontend e backend
- Servidor Express
- Conexão com MySQL
- Cadastro de usuários
- Listagem de usuários
- Edição de usuários
- Exclusão de usuários
